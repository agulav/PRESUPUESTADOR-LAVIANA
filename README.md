# Presupuestador Foton Malaspina

App de cotizaciones para Foton Malaspina · Pehuajó.

## Funciones principales

- Simulador con catálogo completo Foton (Mayo 2026)
- 6 planes de financiación: Contado Efectivo, Pesos, UVA, Dólares, Leasing, Cheques
- Permuta con cálculo automático del saldo a pagar
- Datos cliente uso interno (no salen en el PDF)
- Cotizaciones numeradas (FT-2026-XXXX)
- Vendedores con auto-membrete
- Formularios: Datero PF, Datero PJ, Alta Cliente Corven, Proforma (ARS/USD)
- Snapshots mensuales y anuales
- Exportar a JSON
